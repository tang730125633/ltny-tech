# 储能项目

## 储能类型

### 电化学储能
- **磷酸铁锂**：安全性高，循环寿命长，主流技术路线
- **钠离子电池**：成本低，低温性能好，商业化初期
- **液流电池**：大容量、长时储能优势

### 机械储能
- **抽水蓄能**：最成熟的大规模储能方式
- **压缩空气储能**：新型长时储能技术
- **飞轮储能**：快速响应，适用于调频

## 商业模式

### 电源侧
- 新能源配储（强制配储10%-20%）
- 独立共享储能电站

### 电网侧
- 调峰、调频辅助服务
- 电网安全支撑

### 用户侧
- 工商业峰谷套利
- 需量管理、降低容量费
- 备用电源

## 经济性分析关键

| 指标 | 参考值 |
|:---|:---|
| 度电成本 | 磷酸铁锂 ≤ 0.3元/kWh |
| 循环寿命 | ≥ 6000次 |
| 峰谷价差 | ≥ 0.6元/kWh（用户侧盈利门槛） |
| IRR | ≥ 8%（投资级） |

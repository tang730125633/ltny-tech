# 风电项目

## 项目类型

### 陆上风电
- 三北地区大基地项目
- 中东南部低风速风电
- 分散式风电（乡村振兴"驭风行动"）

### 海上风电
- 近海风电（0-50km）
- 深远海风电（>50km，漂浮式）

## 关键参数

| 参数 | 说明 |
|:---|:---|
| 年等效满发小时数 | 陆上2000-3000h，海上3000-4000h |
| 单机容量 | 陆上6-10MW，海上12-20MW |
| LCOE | 持续下降，部分区域已低于火电 |

## 开发流程

```
1. 风资源评估（测风塔≥1年）→ 2. 可行性研究 → 3. 核准/备案 → 4. 设计 → 5. 施工 → 6. 并网 → 7. 运维
```

## 当前趋势

- 沙戈荒大基地加速开工
- 海上风电规模化发展
- 风电+储能成标配
- 央企划定盈利边界，高质量开发

# 光伏项目开发

## 项目类型

### 分布式光伏
- **工商业屋顶**：利用企业厂房屋顶，自发自用、余电上网
- **户用光伏**：农村家庭屋顶，全额上网或自发自用
- **BIPV**：光伏建筑一体化

### 集中式光伏
- **地面电站**：荒漠、滩涂、采煤沉陷区
- **农光互补**：农业大棚+光伏
- **渔光互补**：水面漂浮式光伏

## 开发流程

```
1. 资源评估 → 2. 可行性研究 → 3. 备案/审批 → 4. 设计 → 5. 建设 → 6. 并网 → 7. 运维
```

## 关键指标

| 指标 | 说明 |
|:---|:---|
| IRR | 内部收益率，一般要求 ≥ 8% |
| LCOE | 度电成本 |
| PR | 系统效率，优质项目 ≥ 82% |
| 容配比 | 组件容量/逆变器容量，1.0-1.3 |

## 政策要点

- 分布式光伏：备案制，无需指标
- 全额上网/自发自用余电上网模式选择
- 国家补贴已退出，依靠平价上网

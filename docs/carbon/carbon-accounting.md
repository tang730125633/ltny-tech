# 碳核算方法学

## 概述

碳核算是指对企业或组织的温室气体排放量进行量化计算的过程，是实现碳达峰、碳中和的基础工作。

## 核算范围（Scope）

### Scope 1：直接排放
- 固定燃烧（锅炉、燃气轮机）
- 移动燃烧（车辆、叉车）
- 工艺过程排放
- 无组织排放

### Scope 2：间接排放（电力）
- 外购电力
- 外购热力/蒸汽
- 外购冷量

### Scope 3：其他间接排放
- 供应链上下游
- 员工通勤
- 产品使用阶段

## 核算方法

### 排放因子法
```
排放量 = 活动数据 × 排放因子 × GWP值
```

### 常用标准
- ISO 14064-1
- GHG Protocol
- 国家发改委24行业指南

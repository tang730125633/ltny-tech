# 碳足迹评估

## 概述

碳足迹是指某一产品在其全生命周期中直接或间接产生的温室气体排放总量。

## 评估范围

### 从摇篮到大门（Cradle-to-Gate）
原材料获取 → 制造加工

### 从摇篮到坟墓（Cradle-to-Grave）
原材料获取 → 制造加工 → 分销 → 使用 → 废弃处理

## 适用标准

- ISO 14067:2018 产品碳足迹
- PAS 2050
- GHG Protocol Product Standard

## 电力行业碳足迹特点

- 光伏组件：生产阶段碳排放较高，运行阶段几乎为零
- 风电设备：全生命周期碳足迹低于化石能源
- 储能系统：需考虑电池生产和回收阶段

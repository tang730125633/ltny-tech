# 能源数字化平台

## 平台架构

```
┌─────────────────────────────────────────┐
│           应用层 (Applications)          │
│  能耗监测 · 负荷预测 · 交易管理 · 碳管理  │
├─────────────────────────────────────────┤
│           平台层 (Platform)              │
│  数据接入 · 数据存储 · 数据分析 · AI引擎  │
├─────────────────────────────────────────┤
│           边缘层 (Edge)                  │
│  数据采集 · 协议转换 · 边缘计算 · 本地存储 │
├─────────────────────────────────────────┤
│           设备层 (Devices)               │
│  电表 · 传感器 · 逆变器 · 储能PCS · 充电桩 │
└─────────────────────────────────────────┘
```

## 核心功能模块

### 1. 能耗监测
- 实时用电数据采集
- 多维度能耗分析
- 异常用电告警
- 报表自动生成

### 2. 负荷预测
- 基于机器学习的负荷预测算法
- 支持日前、日内预测
- 预测精度 ≥ 95%

### 3. 交易管理
- 中长期交易申报
- 现货交易辅助决策
- 收益分析与复盘

### 4. 碳管理
- 碳排放自动核算
- 碳资产管理
- 碳中和路径规划

## 技术栈

- **后端**: Python / Node.js
- **数据库**: PostgreSQL + TimescaleDB
- **时序存储**: InfluxDB
- **消息队列**: MQTT / Kafka
- **前端**: React / Vue
- **AI框架**: PyTorch / TensorFlow

# 电力市场化交易

## 市场结构

```
┌─────────────────────────────────────────┐
│              电力批发市场                │
│  ┌─────────┐  ┌─────────┐  ┌─────────┐ │
│  │中长期市场│  │ 现货市场 │  │ 辅助服务 │ │
│  │(年度/月度│  │(日前/实时│  │         │ │
│  │/月内)   │  │)        │  │         │ │
│  └─────────┘  └─────────┘  └─────────┘ │
└─────────────────────────────────────────┘
                   ↓
┌─────────────────────────────────────────┐
│              电力零售市场                │
│      售电公司 → 工商业用户/居民          │
└─────────────────────────────────────────┘
```

## 交易品种

### 中长期交易
- **年度协商**：锁定年度用电价格
- **月度竞价**：月度价格发现
- **月内交易**：灵活调整偏差

### 现货交易
- **日前市场**：提前一天出清
- **实时市场**：实时平衡
- **结算价格**：按节点边际电价(LMP)

### 辅助服务
- 调频、调峰、备用
- 储能可参与调频市场

## 湖北电力市场特点

- 华中区域电力市场建设加快
- 现货市场已开展试运行
- 新能源参与市场交易比例逐步提高
- 储能可参与调峰、调频市场

## 交易策略

1. **中长期为主，现货为辅** — 控制价格风险
2. **偏差管理** — 预测准确率是关键
3. **储能套利** — 峰谷价差、现货价差

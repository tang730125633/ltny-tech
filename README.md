# 湖北零碳能源科技有限公司 (ltny-tech)

## 公司简介

湖北零碳能源科技有限公司（Hubei Lingtan Energy Technology Co., Ltd.）是一家专注于**双碳领域**和**电力行业**的高科技企业，致力于为企业提供全方位的能源转型解决方案。

---

## 核心业务

### 🔬 双碳咨询
- **碳核算**：企业碳排放数据测算与报告编制
- **碳盘查**：组织层级温室气体排放全面核查
- **碳足迹**：产品全生命周期碳足迹评估与优化

### 💻 能源数字化
- **电力系统软件开发**：配电网智能管理平台、能源监测系统
- **数据平台构建**：能源大数据分析、用电负荷预测
- **智能化解决方案**：物联网接入、AI算法应用

### ⚡ 新能源工程
- **光伏项目**：分布式/集中式光伏电站设计、建设与运维
- **风电项目**：陆上/海上风电场开发、工程管理
- **储能系统**：电化学储能、工商业储能项目集成

### 📊 售电/能源交易
- **电力市场化交易**：参与中长期、现货市场交易
- **能源托管服务**：企业综合能源管理、节能降本方案
- **交易咨询**：电价策略分析、风险控制

## Claude 远程指导 OpenClaw（Git 协作模式）

已内置双向沟通通道：

- 导师下发：`communication/claude-to-xiaol.md`
- OpenClaw 汇报：`communication/xiaol-to-claude.md`
- 协作变更：`communication/changelog.md`
- 远程执行说明：`communication/REMOTE_GUIDANCE_SOP.md`
- 早报模板：`briefings/templates/daily-briefing.md`

建议执行节奏：

1. Claude 更新指导指令并 `git push`
2. OpenClaw `git pull` 后按指令执行并产出到 `briefings/output/`
3. OpenClaw 写回 `xiaol-to-claude.md` 并 `git push`
4. Claude 审阅后将修订版放入 `briefings/reviewed/` 并继续下发下一轮任务

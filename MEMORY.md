## 固化任务执行SOP

### 每日早报（零碳能源行业）
**执行标准**：每天早上 08:00 (GMT+8) 前后先放到“老板办公桌”，审核后再决定是否发送

**执行流程**：
1. **触发机制**：
   - 主：老板办公桌早报任务每天 07:50 自动触发
   - 备：政策监控任务持续巡检，发现新增后更新“当前增量状态”

2. **内容采集**（07:50-07:58）：
   - 国家发改委官网 (ndrc.gov.cn) - 政策动态
   - 国家能源局官网 (nea.gov.cn) - 行业数据、政策发布
   - 华中监管局官网 (hzj.nea.gov.cn) - 区域监管动态

3. **内容上桌**（08:00 前后）：
   - 只放未递送过的新内容
   - 如无新增，明确写“无新增”，禁止旧闻补位
   - 先生成老板办公桌文件，等待老板过目

4. **审核与后续动作**：
   - 老板过目后，再决定是否推送、下载或分派
   - 带官方 PDF 的正式文件优先进入审核池

**失败处理**：
- 如早报桌面文件未成功生成，立即重跑监控和桌面脚本
- 如发现当天无新增，则保持“无新增”状态，不得硬凑
- 如需要正式文件下载或结构化处理，优先调动 Codex / Claude Code 协作
